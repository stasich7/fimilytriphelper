## [activity.mtskheta.daytrip] Мцхета и Джвари
Type: activity

Один из самых удачных выездов из Тбилиси:
- дорога короткая;
- день красивый и спокойный;
- древняя столица Грузии;
- один из самых запоминающихся видов на Джвари.

Это хороший день и для вашей семьи, и для общего выезда с родственниками.

Сюда можно поехать и без экскурсии.

Как добраться:
- самый простой вариант — такси из Тбилиси до Мцхеты;
- более бюджетный вариант — маршрутка из Тбилиси до Мцхеты, а дальше пешком и на такси;
- до Джвари удобнее всего подниматься на такси из Мцхеты, потому что пеший подъем туда уже довольно тяжелый.

Ориентир по расстоянию:
- Тбилиси → Мцхета: около 20 км
- Мцхета → Джвари: около 5 км

Ориентир по цене без экскурсии:
- $11-$15 за машину, за такси Тбилиси → Мцхета
- $4-$7 за машину, за такси Мцхета → Джвари
- $4-$7 за машину, за такси Джвари → Мцхета
- $0.75-$1.50 за человека, за маршрутку Тбилиси → Мцхета
- $0.75-$1.50 за человека, за маршрутку Мцхета → Тбилиси

Как вернуться обратно:
- самый удобный вариант — вызвать такси из Мцхеты в Тбилиси;
- если хочется сэкономить, можно вернуться маршруткой;
- из Джвари лучше сначала спуститься на такси обратно в Мцхету, а уже оттуда ехать в Тбилиси.

Где походить:
- собор Светицховели;
- старые улицы и площадь вокруг центра Мцхеты;
- набережная и виды на слияние рек;
- монастырь Самтавро, если захочется более спокойной прогулки;
- Джвари — это скорее остановка ради видов, чем длинная прогулка.

Если ехать самостоятельно, удобный ритм такой:
- утром выехать из Тбилиси;
- сначала погулять по Мцхете 2-3 часа;
- затем подняться в Джвари;
- после этого вернуться в Мцхету на обед или сразу уехать в Тбилиси.

Ориентир по цене:
- $19.90 за человека, за групповую экскурсию
- от $30 за машину, до 4 пассажиров, если брать частную поездку через GoTrip

## [note.relatives.shared-logistics] Как удобнее размещаться с родственниками
Type: note

Для этой поездки лучше не искать один общий дом на всех.

Более удобная схема:
- сначала выбрать проживание для вашей семьи;
- затем смотреть 1-2 объекта рядом для родственников;
- жить в одной локации или в разумной близости;
- общие дни и ужины планировать отдельно.

Такой подход проще:
- по бронированию;
- по бытовому комфорту;
- по режиму детей и старших взрослых;
- по гибкости всей поездки.
